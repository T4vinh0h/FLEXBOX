📖 5 Tipos de Position

Position define como um elemento HTML é posicionado na página. Cada tipo tem um comportamento diferente e casos de uso específicos.

Tipo Descrição Remove do fluxo? Uso
static Padrão (fluxo normal) Não Padrão
relative Relativo à posição original Não Ajustes finos, referência para absolute
absolute Em relação ao parent com position Sim Sobreposição, menus dropdowns
fixed Fixo em relação ao viewport Sim Menus fixos, botões flutuantes
sticky Combinação de relative e fixed Não Cabeçalhos sticky

🔧 Position Relative

Relative permite ajustar a posição do elemento em relação ao seu local original, sem removê-lo do fluxo.

Sintaxe:

.box {
  position: relative;
  top: 10px; /* 10px para baixo */
  left: 20px; /* 20px para direita */
}

Exemplo Visual:

Sem position (normal)

Normal

Position relative (top: 20px, left: 20px)

Deslocado

🎯 Position Absolute

Absolute posiciona um elemento em relação ao parent mais próximo com position. SAI do fluxo normal (outros elementos não o "enxergam").

⚠️ Regra Importante: Para usar absolute, precisa de um parent com position (geralmente relative).

Sintaxe (DUPLA):

.parent {
  position: relative; /* Cria referência */
}
.child {
  position: absolute; /* Posiciona em relação ao parent */
  top: 0;
  right: 0;
}

Exemplo Visual:

Parent (relative) + Child (absolute no canto superior direito)

Container
ABS

📌 Position Fixed

Fixed deixa o elemento fixo em relação à janela do navegador (viewport). Não se move ao rolar a página.

Sintaxe:

.fixed-element {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

Uso:

Cenário Exemplo
Menu fixo no topo Navegação que sempre fica visível
Botão "voltar ao topo" Aparece ao rolar para baixo
Notificação flutuante Pop-up que não se move

🏗️ Z-index (Camadas)

Z-index define a ordem de empilhamento - qual elemento fica por cima. Maior valor = fica por cima.

Z-index Posição na pilha Exemplo
0 Base Camada inferior
10 Acima de 0 Primeira camada
100 Acima de 10 Segunda camada
1000 Acima de 100 Camada superior
⚠️ Importante: Z-index só funciona em elementos com position (absolute, relative, fixed, sticky)!

✅ Exercício Prático

🎯 Desafio:

  1. Crie um container com position: relative
  2. Dentro, coloque um elemento com position: absolute posicionado no canto
  3. Adicione um badge com z-index: 10 sobreposto
  4. Experimente diferentes z-index para ver a ordem
