📖 Classes vs IDs

CSS oferece dois seletores que permitem estilizar elementos específicos: Classes e IDs. Enquanto estilizar pela tag (como p) afeta TODOS os elementos daquele tipo, Classes e IDs permitem precisão e reutilização.

Característica Classe (class) ID
Símbolo CSS . (ponto) # (hashtag)
Reutilização ✅ Vários elementos podem ter a mesma classe ✅ Pode ter vários IDs, mas cada nome único
Repetição ✅ Pode repetir o mesmo nome ❌ Não pode repetir o mesmo nome
Escopo Grupo de elementos com mesmo estilo Elemento único na página
HTML class="nome-da-classe" id="nome-do-id"

🏷️ Classes

Classes são usadas quando você quer aplicar o mesmo estilo em vários elementos. Imagine que você tem vários cards de produtos - todos podem ter a mesma classe class="card" e herdar o mesmo estilo.

Exemplo - Cards de Produtos:

/* CSS */
.card {
  border: 2px solid blue;
  padding: 20px;
}
/* HTML */
<div class="card">Produto 1</div>
<div class="card">Produto 2</div>
<div class="card">Produto 3</div>

Exemplo Visual:

Cards com Mesma Classe

Produto 1
Produto 2
Produto 3

Quando Usar Classes:

Situação Sim/Não Por quê
Estilo se repete em vários elementos ✅ Sim Classe permite reutilização
Precisa de componentes reutilizáveis ✅ Sim Classes são ideais para componentes
Elemento único na página ❌ Não Use ID

🆔 IDs

IDs são usados para identificar elementos únicos. Cada ID deve ter um nome único na página - você não pode ter dois elementos com id="cabecalho".

Exemplo - Seções da Página:

/* CSS */
#cabecalho {
  background: dark;
  color: white;
}
/* HTML */
<header id="cabecalho">Cabeçalho</header>
<main id="conteudo">Conteúdo</main>
<footer id="rodape">Rodapé</footer>

Quando Usar IDs:

Situação Sim/Não Por quê
Elemento único na página ✅ Sim ID identifica unicamente
Seções principais (header, footer) ✅ Sim Cada seção é única
Estilo se repete ❌ Não Use Classe
Alvos para JavaScript ✅ Sim IDs são fáceis de selecionar com JS

🔲 Bordas (border)

A propriedade border cria bordas ao redor dos elementos. É essencial para destacar elementos, criar containers e dar estrutura ao design.

Sintaxe Resumida:

border: 2px solid blue; /* espessura, estilo, cor */

Estilos de Borda:

Estilo Descrição Exemplo Visual
solid Linha sólida
Solid
dashed Tracejado
Dashed
dotted Pontilhado
Dotted
double Linha dupla
Double

Bordas Específicas:

border-top: 2px solid blue; /* Apenas em cima */
border-right: 2px solid blue; /* Apenas à direita */
border-bottom: 2px solid blue; /* Apenas embaixo */
border-left: 2px solid blue; /* Apenas à esquerda */

Cantos Arredondados (border-radius):

border-radius: 8px; /* Todos os cantos */
border-radius: 10px 0 10px 0; /* Cantos alternados */
border-radius: 50%; /* Círculo/elipse */

✅ Exercício Prático

🎯 Desafio:

  1. Crie 3 cards com a classe class="card"
  2. Aplique bordas e padding nos cards
  3. Dê um ID diferente para o card do meio: id="destaque"
  4. Estilize o ID para ter uma cor de borda diferente
  5. Adicione border-radius para arredondar os cantos
