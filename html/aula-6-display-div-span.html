📖 Tipos de Display

O display define como um elemento HTML se comporta: se ocupa linha inteira, fica lado a lado, aceita width/height, etc. É fundamental para entender o comportamento dos elementos.

Display Ocupa largura total? Aceita width/height? Fica lado a lado? Exemplo
block Sim Sim Não div, p, h1
inline Não Não Sim span, a, strong
inline-block Não Sim Sim Transformado

1️⃣ Display: Block

Elementos de bloco ocupam toda a largura disponível e sempre começam em uma nova linha. Eles "empurram" os elementos seguintes para baixo.

Exemplo Visual:

Elementos Block (DIV)

Div 1 (bloque)
Div 2 (bloque)
Div 3 (bloque)

2️⃣ Display: Inline

Elementos inline ocupam apenas o espaço necessário e ficam lado a lado. Eles não quebram linha e não aceitam width/height.

Exemplo Visual:

Elementos Inline (SPAN)

Span 1

Span 2

Span 3

3️⃣ Display: Inline-Block

Inline-block é o melhor dos dois mundos: fica lado a lado (como inline) e aceita width/height (como block).

Exemplo Visual:

🏷️ DIV vs SPAN

Aspecto DIV SPAN
Tipo Elemento de bloco Elemento inline
Largura Ocupa toda largura Ocupa apenas o necessário
Quebra linha Sim (nova linha) Não
Uso Agrupar conteúdo/seções Estilizar partes de texto

Quando Usar:

Situação Use DIV Use SPAN
Precisa criar seções/blocos ✅ Sim ❌ Não
Quer agrupar vários elementos ✅ Sim ❌ Não
Quer estilizar parte de um texto ❌ Não ✅ Sim
Não quer quebrar a linha ❌ Não ✅ Sim

✅ Exercício Prático

🎯 Desafio:

  1. Crie 3 divs com conteúdo
  2. Transforme em inline-block para ficarem lado a lado
  3. Defina width e height nas divs
  4. Use <span> para destacar palavras em um texto

🎓 Resumo da Aula

Conceito Comportamento Uso
Display Block Ocupa toda largura, nova linha Estrutura principal
Display Inline Lado a lado, sem width/height Partes de texto
Display Inline-Block Lado a lado, aceita width/height Layouts flexíveis
DIV Elemento de bloco Agrupar conteúdo
SPAN Elemento inline Estilizar texto
