▶ O QUE É HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas web.

• HTML não é uma linguagem de programação

• HTML é uma linguagem de marcação (usa tags para estruturar conteúdo)

• Todo site que você acessa usa HTML

Exemplo: Quando você acessa um site, o navegador lê o código HTML e mostra a página formatada na tela.

▶ O QUE É CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML.

• CSS define cores, tamanhos, fontes, espaçamentos

• Deixa as páginas bonitas e organizadas

• Trabalha junto com o HTML

Resumo: - HTML = Estrutura e conteúdo (texto, imagens, links) - CSS = Aparência e estilo (cores, tamanhos, posicionamento)

▶ O QUE É O VSCODE?

Visual Studio Code (VSCode) é um editor de código gratuito e poderoso.

Por que usar o VSCode?

• Gratuito e leve

• Destaca a sintaxe do código com cores

• Autocompleta código

• Tem extensões úteis

• Funciona em Windows, Mac e Linux

▶ COMO CRIAR UM ARQUIVO HTML NO VSCODE?

Passo a passo:

1. Abra o VSCode

2. Clique em File → New File (ou Ctrl + N)

3. Salve o arquivo com extensão .html (exemplo: index.html)

4. Pronto! Agora pode começar a escrever HTML

▶ O QUE É UM ARQUIVO .HTML?

Um arquivo .html é um arquivo de texto com código HTML que o navegador consegue ler e exibir.

Características:

• Extensão: .html ou .htm

• Pode ser aberto em qualquer navegador (Chrome, Firefox, Edge)

• Pode ser editado em qualquer editor de texto

Exemplo de nome de arquivo: - index.html (página principal) - sobre.html - contato.html

▶ ESTRUTURA BÁSICA DO HTML  ⚠️ Atalho: ! + Enter

No VSCode, digite ! (exclamação) e aperte Enter. Ele cria automaticamente esta estrutura:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>

Entendendo cada parte:

<!DOCTYPE html> - Diz ao navegador que este é um documento HTML5 (versão mais recente).

<html lang="en"> - Tag que envolve todo o conteúdo da página. - lang="en" define o idioma (mude para "pt-BR" para português)

<head> - Cabeçalho com informações que não aparecem na página (metadados, título da aba, links CSS).

<meta charset="UTF-8"> - Define a codificação de caracteres (permite acentos e caracteres especiais).

<meta name="viewport" content="width=device-width, initial-scale=1.0"> - Torna o site responsivo em celulares e tablets.

<title>Document</title> - Título que aparece na aba do navegador.

<body> - Corpo da página - tudo que você escrever aqui aparece na tela.

▶ TAGS HTML - TÍTULOS

HTML tem 6 níveis de títulos, de h1 (maior) até h6 (menor).

Título Principal (h1)

Subtítulo (h2)

Título de Seção (h3)

Título Menor (h4)

Título Ainda Menor (h5)
Título Mínimo (h6)

Sintaxe:

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Título de Seção</h3>
<h4>Título Menor</h4>
<h5>Título Ainda Menor</h5>
<h6>Título Mínimo</h6>

Dicas: • Use h1 para o título principal da página (apenas um por página) • Use h2 para seções importantes • Use h3 para subseções • Não pule níveis (não vá direto de h1 para h4)

▶ TAG DE PARÁGRAFO

A tag <p> é usada para criar parágrafos de texto.

Bem-vindo ao meu site

Este é o primeiro parágrafo do site.

Este é o segundo parágrafo com mais informações.

Sintaxe:

<p>Este é um parágrafo de texto.</p>
<p>Este é outro parágrafo.</p>

Dica: O navegador automaticamente coloca espaço entre parágrafos.

▶ TAG DE IMAGEM

A tag <img> é usada para inserir imagens na página.

Sintaxe:

<img src="caminho-da-imagem.jpg" alt="Descrição da imagem">

Atributos importantes:

src (source = fonte) - Caminho da imagem - Pode ser local: "foto.jpg" - Ou da internet: "https://site.com/imagem.jpg"

alt (alternative = alternativo) - Texto alternativo caso a imagem não carregue - Usado por leitores de tela (acessibilidade) - Descreve o que tem na imagem

Exemplos:

<!-- Imagem local (na mesma pasta) -->
<img src="logo.png" alt="Logo da empresa">

<!-- Imagem da internet -->
<img src="https://example.com/foto.jpg" alt="Foto de exemplo">

<!-- Imagem em uma subpasta -->
<img src="imagens/banner.jpg" alt="Banner do site">

Importante: A tag <img> não tem tag de fechamento (é uma tag auto-fechada) - Correto: <img src="foto.jpg" alt="Foto"> - Errado: <img src="foto.jpg" alt="Foto"></img>

▶ TAG DE LINK

A tag <a> (anchor = âncora) cria links clicáveis.

Confira meu perfil:

LinkedIn

Sintaxe:

<a href="destino">Texto do link</a>

Atributo principal:

href (hyperlink reference) - Para onde o link vai levar - Pode ser outra página do site - Pode ser um site externo

Exemplos:

<!-- Link para outra página do site -->
<a href="sobre.html">Sobre nós</a>

<!-- Link para site externo -->
<a href="https://www.google.com">Ir para o Google</a>

<!-- Link que abre em nova aba -->
<a href="https://www.google.com" target="_blank">Google (nova aba)</a>

Dica: Use target="_blank" para abrir o link em uma nova aba.

▶ TAG DE QUEBRA DE LINHA

A tag <br> cria uma quebra de linha (pula para a próxima linha).

Rua das Flores, 123
Centro - São Paulo/SP
CEP: 01234-567

Sintaxe:

Primeira linha<br>
Segunda linha<br>
Terceira linha

Quando usar: Para endereços, Para poemas, Quando precisa quebrar linha dentro de um parágrafo

A tag <br> não tem tag de fechamento

▶ TAG DE LINHA HORIZONTAL

A tag <hr> cria uma linha horizontal (separador visual).

Seção 1

Conteúdo da primeira seção.

Seção 2

Conteúdo da segunda seção.

Sintaxe:

<hr>

Quando usar: • Para separar seções do conteúdo • Para criar divisões visuais

A tag <hr> não tem tag de fechamento

<!DOCTYPE html>

    <p>Este é meu primeiro site criado com HTML!</p>
    
    <h2>Sobre Mim</h2>
    <p>Meu nome é João e estou aprendendo desenvolvimento web.</p>
    <p>Estou muito animado com esta jornada de aprendizado.</p>
    
    <hr>
    
    <h2>Minha Foto</h2>
    <img src="foto.jpg" alt="Minha foto de perfil">
    
    <h2>Links Úteis</h2>
    <p>Confira meu perfil:</p>
    <a href="https://www.linkedin.com" target="_blank">LinkedIn</a>
    
    <hr>
    
    <h3>Contato</h3>
    <p>
        Email: joao@email.com<br>
        Telefone: (11) 98765-4321<br>
        São Paulo - SP
    </p>
</body>
</html>

▶ RESUMO DAS TAGS APRENDIDAS

Tag Descrição Tem Fechamento?
<h1> até <h6> Títulos (6 níveis) Sim
<p> Parágrafo Sim
<img> Imagem Não
<a> Link Sim
<br> Quebra de linha Não
<hr> Linha horizontal Não

▶ DICAS IMPORTANTES

• Sempre salve o arquivo com extensão .html

• Use lang="pt-BR" para sites em português

• Altere o <title> para um nome descritivo

• Todo conteúdo visível vai dentro do <body>

• Sempre adicione o atributo alt nas imagens

• Teste seu código abrindo o arquivo no navegador

Atalho útil: No VSCode, digite ! + Enter para criar a estrutura HTML básica!
