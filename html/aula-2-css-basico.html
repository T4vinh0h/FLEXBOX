📖 O que é CSS?

CSS (Cascading Style Sheets) é a linguagem que estiliza páginas HTML. Se HTML cria a estrutura e o conteúdo, CSS é responsável pela aparência visual - cores, fontes, espaçamentos, layout e muito mais. Sem CSS, as páginas HTML seriam apenas texto branco sobre fundo preto, sem formatação nenhuma.

Analogia Simples:

Elemento Papel Analogia
HTML Estrutura e conteúdo Esqueleto do corpo
CSS Estilização visual Roupas e maquiagem
� Regra de Ouro: HTML cria a estrutura, CSS deixa bonito!

🔧 Sintaxe CSS

Para estilizar elementos HTML, você escreve regras CSS que seguem uma estrutura específica: seletor, propriedade e valor.

Anatomia de uma Regra CSS:

p {
  color: blue;
  font-size: 16px;
}
Parte Descrição Exemplo
p Seletor - qual elemento estilizar Todos os parágrafos
{ } Bloco de declarações Abrir e fechar chaves
color: blue; Declaração - propriedade + valor Cor do texto = azul
color Propriedade - o que alterar Cor do texto
blue Valor - como alterar Cor azul
; Ponto e vírgula - separa declarações Final de cada propriedade
⚠️ Regras Essenciais:
  • Sempre termine declarações com ;
  • Abrir e fechar chaves { }
  • Escrever propriedades em minúsculo
  • Separação entre propriedade e valor com :

📋 Os 3 Tipos de CSS

CSS pode ser aplicado de três formas diferentes: Inline (na tag HTML), Interno (no cabeçalho do HTML) e Externo (em arquivo separado). Cada método tem seus usos específicos e vantagens.

Comparativo dos 3 Tipos:

Tipo Onde fica Vantagens Desvantagens Recomendado?
Inline Dentro da tag HTML Rápido para testes, afeta só aquela tag Difícil manter, código repetitivo Só testes
Interno Tag <style> no <head> Melhor que inline, estilo centralizado Só funciona na página, não reaproveitável Às vezes
Externo Arquivo .css separado Reutilizável, organizado, profissional Precisa de dois arquivos SIM! ✅

1️⃣ CSS Inline

Quando você estiliza um elemento diretamente na tag HTML usando o atributo style. Isso aplica CSS apenas àquele elemento específico.

Sintaxe:

<p style="color: red; font-size: 20px;">Texto em vermelho</p>

Exemplo Visual:

Exemplo de CSS Inline

Este texto tem CSS inline aplicado!

Vantagens e Desvantagens:

Aspecto Avaliação
Velocidade de testes ✅ Excelente para testes rápidos
Manutenção ❌ Difícil de manter e atualizar
Reutilização ❌ Não pode reaproveitar o estilo
Profissionalismo ❌ Não é prática profissional

2️⃣ CSS Interno (tag <style>)

Quando você escreve CSS dentro do HTML, na tag <style> localizada dentro do <head>. O CSS fica disponível para toda a página, mas não para outras páginas.

Sintaxe:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Meu Site</title>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>

Vantagens e Desvantagens:

Aspecto Avaliação
Centralização ✅ Estilo em um lugar
Escopo ❌ Só funciona na página HTML
Reutilização ❌ Não pode usar em outras páginas
Tamanho do arquivo ❌ HTML fica muito grande

3️⃣ CSS Externo (arquivo .css separado) ✅

Quando você escreve CSS em um arquivo separado com extensão .css e o vincula ao HTML usando a tag <link>. Este é o método RECOMENDADO para projetos profissionais.

Estrutura:

Arquivo Conteúdo
style.css Código CSS
index.html HTML que referencia o CSS

Passo 1: Criar arquivo CSS:

/* Arquivo: style.css */
p {
  color: blue;
  font-size: 16px;
}
h1 {
  color: red;
}

Passo 2: Vincular no HTML:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Meu Site</title>
  <link rel="stylesheet" href="style.css">
</head>

Exemplo Visual:

Texto estilizado via CSS Externo

Este parágrafo está estilizado via arquivo CSS externo!

Vantagens do CSS Externo:

Vantagem Explicação
✅ Reutilizável Pode usar o mesmo CSS em várias páginas
✅ Organizado HTML e CSS separados, mais limpo
✅ Fácil manutenção Mudanças em um lugar afetam tudo
✅ Melhor performance Navegador pode cachear o arquivo
✅ Profissional Padrão da indústria

🎯 Seletores CSS

Seletores CSS determinam quais elementos HTML serão estilizados. Quando você estiliza uma tag (como p, h1, li), o estilo é aplicado em TODOS os elementos daquele tipo na página.

Seletores Comuns:

Seletor Alvo Exemplo de CSS Afeta
h1 Todos os h1 color: blue; Todos os títulos h1
p Todos os parágrafos color: green; font-size: 18px; Todos os parágrafos
ul Todas as listas UL color: purple; Todas as listas não ordenadas
li Todos os itens de lista font-size: 14px; Todos os LI
⚠️ Importante: Quando você estiliza uma tag, o estilo afeta TODOS os elementos daquele tipo na página. Para estilizar elementos específicos, você deve usar Classes (aula 4).

✅ Exercício Prático

🎯 Desafio:

  1. Crie um arquivo index.html básico
  2. Crie um arquivo style.css na mesma pasta
  3. No CSS, estilize h1 para ser azul com tamanho 32px
  4. Estilize p para ser cinza escuro com tamanho 16px
  5. Vincule o CSS no HTML usando a tag <link>
  6. Abra o HTML no navegador e veja o resultado
  7. Experimente adicionar CSS inline em um parágrafo para ver a diferença
