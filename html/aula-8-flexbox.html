📦 Aula 8 - Flexbox

🎯 Objetivos da Aula

  • Entender o que é Flexbox
  • Aprender display: flex
  • Conhecer flex-direction
  • Aprender justify-content
  • Aprender align-items
  • Entender flex-wrap e gap

📖 O que é Flexbox?

Flexbox (Flexible Box Layout) é um sistema de layout que facilita organizar, alinhar e distribuir elementos em uma página.

Por que usar?

  • ✅ Facilita alinhar elementos horizontal e verticalmente
  • ✅ Distribui espaço automaticamente
  • ✅ Elementos responsivos (se adaptam ao tamanho da tela)
  • ✅ Resolve problemas que antes exigiam truques complicados
💡 Dica: Temos um exemplo prático completo de Flexbox no arquivo index.html deste projeto!

🔧 Como funciona?

Flexbox trabalha com dois conceitos:

  • Container (Pai) - elemento com display: flex
  • Items (Filhos) - elementos dentro do container
<div class="container"> <!-- Container/Pai -->
  <div>Item 1</div> <!-- Item/Filho -->
  <div>Item 2</div> <!-- Item/Filho -->
  <div>Item 3</div> <!-- Item/Filho -->
</div>
.container {
  display: flex; /* Ativa o flexbox */
}

✨ Display: flex

Transforma um elemento em um flex container.

.container {
  display: flex;
}

O que acontece:

  • ✅ Todos os filhos diretos viram flex items
  • ✅ Por padrão, ficam lado a lado (em linha)
  • ✅ Ocupam apenas o espaço necessário

🔄 Flex-direction - Direção dos itens

Define se os itens ficam em linha ou coluna.

.container {
  display: flex;
  flex-direction: row; /* Padrão */
}

Valores:

  • row (padrão) - Linha (lado a lado, da esquerda para direita)
  • row-reverse - Linha invertida (direita para esquerda)
  • column - Coluna (um embaixo do outro)
  • column-reverse - Coluna invertida (de baixo para cima)

↔️ Justify-content - Alinhamento no eixo principal

Distribui os itens ao longo do eixo principal (horizontal se row, vertical se column).

.container {
  display: flex;
  justify-content: center; /* Padrão */
}

Valores principais:

  • flex-start (padrão) - Início
  • flex-end - Final
  • center - Centro
  • space-between - Espaço entre (extremos grudados)
  • space-around - Espaço ao redor
  • space-evenly - Espaço uniforme

↕️ Align-items - Alinhamento no eixo transversal

Alinha os itens no eixo transversal (vertical se row, horizontal se column).

.container {
  display: flex;
  align-items: center;
}

Valores principais:

  • stretch (padrão) - Estica para preencher
  • flex-start - Início
  • flex-end - Final
  • center - Centro

🔃 Flex-wrap - Quebra de linha

Define se os itens devem quebrar linha quando não couberem.

.container {
  display: flex;
  flex-wrap: wrap; /* Quebra linha */
}

Valores:

  • nowrap (padrão) - Não quebra
  • wrap - Quebra linha
  • wrap-reverse - Quebra linha invertida

📏 Gap - Espaçamento entre itens

Define o espaço entre os itens (substitui margin).

.container {
  display: flex;
  gap: 20px;
}
💡 Dica: gap é melhor que margin para espaçamento em flexbox!

🎯 Casos Práticos Comuns

1. Centralizar elemento (vertical e horizontal)

.container {
  display: flex;
  justify-content: center; /* Horizontal */
  align-items: center; /* Vertical */
  height: 300px;
}

2. Navbar

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

3. Cards lado a lado

.cards-container {
  display: flex;
  gap: 20px;
  flex-wrap: wrap;
}
.card {
  flex: 1; /* Todos crescem igualmente */
}

✅ Exercício Prático

🎯 Tente criar:

  1. Crie um container com 3 itens
  2. Use display: flex para colocá-los lado a lado
  3. Centralize os itens com justify-content: center
  4. Adicione gap: 20px entre os itens
  5. Experimente flex-direction: column
  6. Teste diferentes combinações de justify-content e align-items

🎓 Resumo da Aula

  • ✅ Flexbox = sistema de layout flexível
  • ✅ display: flex = ativa o flexbox
  • ✅ flex-direction = direção (row, column)
  • ✅ justify-content = alinha no eixo principal
  • ✅ align-items = alinha no eixo transversal
  • ✅ flex-wrap = quebra de linha
  • ✅ gap = espaço entre itens
  • ✅ justify-content: center + align-items: center = centraliza elemento
  • ✅ justify-content: space-between = distribui espaço
