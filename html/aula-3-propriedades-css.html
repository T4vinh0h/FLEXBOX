📖 Visão Geral das Propriedades

As propriedades CSS são como as "ferramentas" que você usa para estilizar elementos. Assim como um pintor tem pincéis de diferentes tamanhos e cores, você tem propriedades CSS para controlar cores, fontes, espaçamentos e muito mais. Nesta aula, focaremos nas propriedades mais usadas e essenciais.

1️⃣ Color (cor do texto)

A propriedade color define a cor do texto de um elemento. É uma das propriedades mais básicas e essenciais, pois o contraste entre texto e fundo é crucial para legibilidade e estética.

Sintaxe:

p {
  color: red; /* Nome da cor */
  color: #FF0000; /* Código hexadecimal */
  color: rgb(255, 0, 0); /* RGB */
}

Formas de Definir Cores:

Tipo Sintaxe Exemplos Quando Usar
Nomes de Cores Nome em inglês red, blue, green, orange Cores básicas comuns
Hexadecimal #RRGGBB #FF0000, #00FF00, #0000FF Cores específicas e precisas
RGB rgb(R, G, B) rgb(255, 0, 0), rgb(0, 255, 0) Cores dinâmicas e calculadas

Exemplos Visuais:

Diferentes Cores de Texto

Vermelho (#e94560)
Azul (#667eea)
Verde (#2ecc71)
Laranja (#f39c12)

2️⃣ Background-color (cor de fundo)

A propriedade background-color define a cor de fundo de um elemento. É essencial para criar contraste, seções coloridas e hierarquia visual.

Sintaxe:

body {
  background-color: lightgray;
}
p {
  background-color: yellow;
}
h1 {
  background-color: #333333;
  color: white; /* Texto branco no fundo escuro */
}

Exemplos Visuais:

Diferentes Cores de Fundo

Fundo cinza claro
Fundo azul com texto branco
Fundo verde
💡 Dica Importante: Sempre verifique o contraste entre texto e fundo. O W3C recomenda razão de contraste mínimo de 4.5:1 para texto normal.

3️⃣ Font-size (tamanho da fonte)

A propriedade font-size define o tamanho do texto em pixels. É fundamental para criar hierarquia visual - títulos maiores que subtítulos, subtítulos maiores que corpo do texto.

Sintaxe:

h1 {
  font-size: 32px; /* Títulos grandes */
}
p {
  font-size: 16px; /* Tamanho padrão */
}
li {
  font-size: 14px; /* Texto menor */
}

Tamanhos Comuns e Uso:

Tamanho Uso Típico Resultado Visual
12px Notas de rodapé, avisos pequenos Texto muito pequeno
14px Textos secundários Texto pequeno
16px Texto normal (padrão) Texto normal
20px Subtítulos Texto médio
32px Títulos principais (H1) Texto grande
48px+ Hero titles, chamadas de destaque Muito grande
📝 Unidade: px (pixels) = tamanho fixo em pixels. Para design responsivo, outras unidades como rem, em, % podem ser mais úteis (veja aula 9).

4️⃣ Font-family (tipo de fonte)

A propriedade font-family define qual fonte (tipo de letra) será usada. A escolha da fonte afeta dramaticamente a personalidade e legibilidade do seu design.

Sintaxe:

body {
  font-family: Arial, sans-serif;
}
h1 {
  font-family: 'Times New Roman', serif;
}

Categorias de Fontes:

Categoria Exemplos Característica Exemplo Visual
Sans-serif Arial, Helvetica, Verdana Sem serifa, moderna e limpa Arial Sans-serif
Serif Times New Roman, Georgia Com serifa, tradicional Times Serif
Monospace Courier New, monospace Monoespaçada, técnica Courier Mono
⚠️ Regra Importante: Sempre coloque uma fonte alternativa! Se a primeira não estiver disponível, o navegador usa a próxima. Ex: font-family: Arial, sans-serif;

5️⃣ Font-weight (peso da fonte)

A propriedade font-weight define se o texto é negrito ou não, e sua espessura. É crucial para criar hierarquia e destaque visual.

Sintaxe:

h1 {
  font-weight: bold; /* Negrito */
}
p {
  font-weight: normal; /* Normal */
}

Valores Disponíveis:

Valor Descrição Resultado Visual
normal Peso normal (padrão) Texto normal
bold Negrito Texto negrito
300 Muito fino Texto fino
500 Médio Texto médio
700 Forte Texto forte
900 Muito forte Texto muito forte

✅ Exercício Prático

🎯 Desafio:

  1. Crie uma página com título, subtítulo e parágrafos
  2. Use color para diferentes cores para cada nível de hierarquia
  3. Use background-color para criar seções coloridas
  4. Use font-size para criar hierarquia (H1 grande, H2 médio, P normal)
  5. Use font-family para experimentar diferentes fontes
  6. Use font-weight para destacar textos importantes
  7. Verifique o contraste entre texto e fundo

🎓 Resumo da Aula

Propriedade Função Valores Comuns Dica
color Cor do texto nomes, hex, RGB Ver contraste com fundo
background-color Cor de fundo nomes, hex, RGB Crie contraste com texto
font-size Tamanho da fonte px, rem, em Crie hierarquia visual
font-family Tipo de fonte Arial, Times, etc. Use alternativas
font-weight Peso da fonte normal, bold, 100-900 Crie destaque visual
