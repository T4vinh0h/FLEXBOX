📖 Formulários HTML

Formulários são usados para coletar dados do usuário. São essenciais para cadastros, login, contato e interatividade.

Tag <form>:

<form action="processar.php" method="POST">
  <!-- Campos do formulário -->
</form>
Atributo Função Valor
action Para onde enviar dados URL do arquivo
method Como enviar GET (visível) ou POST (oculto)

📥 Inputs - Tipos e Exemplos

Type Descrição Exemplo Visual
text Texto simples
email Email
password Senha
number Número
checkbox Caixa de seleção
radio Botão de opção
date Data

🏷️ Labels

A tag <label> identifica o campo e melhora a acessibilidade.

<label for="nome">Nome:</label>
<input type="text" id="nome">
💡 Dica: Labels tornam o clique mais fácil (não precisa mirar no input) e são essenciais para leitores de tela.

� Tabelas HTML

Tabelas organizam dados em linhas e colunas.

Estrutura:

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>João</td>
    <td>25</td>
  </tr>
</table>

Exemplo Visual:

Tabela Exemplo

Nome Idade Cidade
João 25 São Paulo
Maria 30 Rio de Janeiro

Estrutura Semântica:

Tag Função
<thead> Cabeçalho (títulos das colunas)
<tbody> Corpo (dados)
<tr> Linha
<th> Célula de cabeçalho
<td> Célula de dados

Colspan e Rowspan:

<th colspan="2">Mesclar 2 colunas</th>
<td rowspan="2">Mesclar 2 linhas</td>

✅ Exercício Prático

🎯 Desafio:

  1. Crie formulário com nome, email, senha
  2. Use labels com for e id
  3. Crie tabela com 3 linhas e 3 colunas
  4. Use colspan para mesclar células

🎓 Resumo da Aula

Conceito Descrição
<form> Envolve formulário (action, method)
<input> Campos de entrada (text, email, password, etc.)
<label> Rótulo do campo (acessibilidade)
<table> Tabela
<tr> Linha
colspan/rowspan Mesclar colunas/linhas
