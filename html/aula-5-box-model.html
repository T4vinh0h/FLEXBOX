📖 O que é Box Model?

Todo elemento HTML é uma "caixa" com 4 camadas que definem como ele ocupa espaço na página. Entender o Box Model é fundamental para criar layouts precisos e controlar espaçamentos.

As 4 Camadas:

Camada Descrição Ação
Content O conteúdo real (texto, imagem) Acontece no centro
Padding Espaço entre conteúdo e borda Empurra conteúdo para dentro
Border Borda ao redor do padding Define limite visual
Margin Espaço fora da borda Empurra elementos para fora

🔵 Padding (espaçamento interno)

Padding é o espaço entre o conteúdo e a borda. Ele "empurra" o conteúdo para dentro, criando "respiro" interno.

Sintaxe:

/* Todos os lados iguais */
padding: 20px;
/* Vertical e Horizontal */
padding: 20px 10px;
/* Cada lado específico */
padding: 10px 15px 20px 25px;

Exemplo Visual:

Sem Padding vs Com Padding

Sem padding
Com padding (20px)

🟢 Margin (espaçamento externo)

Margin é o espaço entre elementos (fora da borda). Ela "empurra" elementos para longe uns dos outros, criando espaçamento entre elementos diferentes.

Sintaxe:

margin: 20px; /* Todos os lados */
margin: 20px 0; /* Vertical */
margin: 0 auto; /* Centraliza horizontalmente */

Comparativo Padding vs Margin:

Padding (interno) vs Margin (externo)

Padding interno
Margin externo
Aspecto Padding Margin
Afeta Espaço interno Espaço externo
Cor de fundo Afeta (tem cor) Não afeta (transparente)
Ação Empurra conteúdo para dentro Empurra elementos para fora
Valores negativos Não aceita Aceita

⚙️ Box-sizing

A propriedade box-sizing define como o navegador calcula o tamanho total do elemento. O valor border-box é essencial para facilitar cálculos de layout.

Valor Descrição Exemplo
content-box (padrão) Padding e border aumentam o tamanho width: 200px + padding: 20px = 240px
border-box (RECOMENDADO) Padding e border inclusos no tamanho width: 200px (já inclui padding)

Reset Recomendado:

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
💡 Por que border-box? Com border-box, o tamanho que você define é o tamanho total. Se você diz que um elemento tem 200px, ele vai ter 200px no total, incluindo padding e border. Muito mais fácil para layout!

✅ Exercício Prático

🎯 Desafio:

  1. Crie 3 cards
  2. Adicione padding: 20px nos cards
  3. Adicione margin: 20px 0 entre os cards
  4. Adicione bordas
  5. Teste com e sem box-sizing: border-box
